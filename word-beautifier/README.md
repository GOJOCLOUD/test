# Word Beautifier

自动美化 Word 文档的字体、字号、行距等样式。



| 文件                           | 作用                                              |

| ---------------------------- | ----------------------------------------------- |

| \*\*web/app.py\*\*               | Flask 后端，负责接收上传、调用 `beautifier.py` 处理、返回下载链接。   |

| \*\*web/templates/index.html\*\* | 网页结构：标题、三个按钮（选择文件 / 开始美化 / 下载结果）和输出路径输入框。       |

| \*\*web/static/style.css\*\*     | 页面样式：白底、居中卡片布局、三按钮横排。                           |

| \*\*web/static/script.js\*\*     | 前端逻辑：显示选中文件、发送上传请求、执行美化、选择输出目录、下载文件。            |

| \*\*beautifier.py\*\*            | 核心逻辑：用 `python-docx` 调整 Word 格式（字体、字号、行距、标题识别）。 |

| \*\*config/default.json\*\*      | 格式模板（字体、字号、行距配置）。                               |

| \*\*input/\*\*                   | 临时存放上传文件。                                       |

| \*\*output/\*\*                  | 默认存放生成的美化结果。                                    |
